# Runtime do Servidor Local

Esta pasta sera usada para rodar o servidor local do Aetheria Online.
No futuro, o arquivo do Paper deve ficar aqui com o nome paper.jar.

Regras de versionamento:
- Mundos, logs e outros arquivos de runtime nao devem ser versionados.
- Plugins .jar nao devem ser enviados ao Git.

Versao alvo:
- Prioridade: Paper 26.2 (se estiver estavel no site oficial do PaperMC).
- Fallback: Paper 26.1.2 (se 26.2 ainda nao estiver estavel).
